# Personas y CuentasBancarias Skillnest 

## Descripción

Este proyecto fue desarrollado en Java con el objetivo de practicar los fundamentos de la Programación Orientada a Objetos (POO), implementando clases, encapsulamiento, constructores, atributos estáticos y relaciones entre objetos.

La aplicación simula un sistema básico de gestión de cuentas bancarias, donde cada cuenta está asociada a una persona titular y permite realizar operaciones de depósito y retiro.

---

## Objetivos

- Utilizar clases para estructurar y gestionar datos de manera efectiva.
- Proteger los datos internos de las clases usando atributos privados y métodos públicos para su acceso y modificación.
- Practicar la instanciación de clases y el uso de diferentes tipos de constructores.
- Gestionar listas de objetos para operar con múltiples instancias.
- Desarrollar métodos que ejecuten funciones específicas relativas a los atributos de los objetos.
- Aplicar métodos que modifiquen el estado de los objetos y realicen cálculos necesarios.
- Hacer uso de atributos estáticos para llevar control sobre las instancias creadas de la clase.
- Verificar la implementación con pruebas que aseguren el correcto funcionamiento de la clase y sus métodos.

---

## Funcionalidades Implementadas

- Creación de personas.
- Creación de cuentas bancarias.
- Asociación entre cuentas y titulares.
- Depósitos.
- Retiros con validación de saldo.
- Generación automática de número de cuenta.
- Registro automático de todas las cuentas creadas.
- Impresión de información individual y general.
